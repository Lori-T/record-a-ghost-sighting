Accessibility statement for Record a ghost sighting

This website is run by the Department for Supernatural Affairs*. It has been created as part of an accessibility training exercise and intentionally contains accessibility issues.

The purpose of this site is to help learners identify, experience and fix common accessibility problems in a realistic environment. It should not be used as an example of an accessible digital service.

*not a real government department.

How accessible this website is

This site is not fully accessible. The following are examples of issues that have been deliberately included:

  • missing or incorrect form labels
  • focus indicators removed or inconsistent
  • colour contrast failures
  • heading hierarchy errors
  • links that lack clear purpose
  • interactive elements that cannot be reached with a keyboard
  • missing image alt text
  • tables without captions or summaries

Purpose of this training site

This service has been designed as part of a training resource to build awareness and confidence in testing and identifying accessibility barriers. Every page intentionally fails one or more WCAG 2.2 AA success criteria.

Feedback and contact information

Because this is a training resource, we don't intend to resolve these intentional issues. However, you can share feedback or suggest improvements by emailing lorithomson@hotmail.co.uk.

Preparation of this accessibility statement

This statement was prepared on 31 October 2025. It was last reviewed on 31 October 2025.
