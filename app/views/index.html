Welcome to the Department for Supernatural Affairs training prototype. This eerie GOV.UK-style service has been intentionally built with accessibility issues to help you test, identify and understand accessibility barriers.

Each page of this service contains one or more accessibility issues (and perhaps a lurking ghost 👻).
